# Assessment System Drop Down Component

##  Usage

```html
<!-- basic usage -->
    <drop-down class="myDropdown">
        <div value="one" slot="options" class="option-item">
            one
        </div>
        <div value="two" slot="options" class="option-item">
            two
        </div>
        <div value="three" slot="options" class="option-item">
            three
        </div>
    </drop-down>
```
